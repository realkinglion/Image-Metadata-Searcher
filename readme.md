# 画像メタ情報検索くん v5.2 DnD Final 取扱説明書（完全版）

## 📖 1. はじめに

この度は、「**画像メタ情報検索くん v5.2 DnD Final**」をご利用いただき、誠にありがとうございます。

本アプリケーションは、AI生成画像のプロンプト（画像を作るための「呪文」や「命令文」）をもとに、膨大な画像コレクションから目的の画像を瞬時に検索できる高性能ツールです。

### 🆕 v5.2 DnD Finalの革新的機能

- **📁 複数ファイル一括ドラッグ** - 選択した複数画像をまとめて操作
- **🎨 進捗ダイアログ** - ファイル操作の進行状況をリアルタイム表示
- **SQLiteデータベース**による高速キャッシュシステム
- **並列処理**によるサムネイル生成の劇的高速化  
- **予測キャッシング**でページ移動が瞬間的に
- **NovelAI専用機能**で生成画像を素早く抽出
- **リアルタイム監視**で新規ファイルを自動検出
- **階層化プロンプト**対応で複雑な構造にも対応

## 📋 2. システム要件（v5.2 DnD版）

### 基本要件
- **OS**: Windows 10/11 推奨（Windows 8.1以降で動作）
- **Python**: 3.8以降**必須**（3.9以降強く推奨）
- **メモリ**: 8GB以上推奨（4GB最小、大量画像処理時は16GB推奨）
- **ディスク容量**: 500MB以上の空き容量（キャッシュDB用）
- **権限**: 通常のユーザー権限（管理者権限不要）

### 推奨環境
- **CPU**: 4コア以上（並列処理最適化のため）
- **ストレージ**: SSD推奨（データベースアクセス高速化）
- **ネットワーク**: ローカルファイル推奨（ネットワークドライブは性能低下）

### 新規依存関係（v5.2で追加）
- **tkinterdnd2**: ドラッグ&ドロップ機能**（新規必須）**
- **Pillow-SIMD**: 画像処理の高速化（通常のPillowでも動作）
- **watchdog**: リアルタイムファイル監視
- **exifread**: EXIFメタデータ読み取り

### 🔍 5.2 検索の基本手順

#### 手順1：検索対象フォルダを指定
1. **従来の方法**: 「**フォルダ選択**」ボタンをクリック
2. **🆕 新方法**: フォルダを**検索フォルダ入力欄にドラッグ&ドロップ**
3. 選択後、リアルタイム監視が自動開始されます

#### 手順2：検索キーワードを入力
**プロンプト例**：
- `1girl` - 女の子が描かれた画像
- `blue hair` - 青い髪の画像  
- `cat ears` - 猫耳の画像
- `masterpiece, best quality` - 高品質指定
- `school uniform` - 制服

#### 手順3：検索オプションを設定
- **AND検索**: すべてのキーワードを含む画像のみ
- **OR検索**: いずれかのキーワードを含む画像
- **一部一致/完全一致**: 検索の厳密さを調整
- **ネガティブ含む**: ネガティブプロンプトも検索対象に
- **サブフォルダも検索**: 子フォルダ内も検索

#### 手順4：検索実行
- 青い「**検索スタート**」ボタンをクリック
- または、キーワード入力欄で **Enterキー**
- 進行中は「**キャンセル**」ボタンで中断可能

### 🎯 5.3 革新的なドラッグ&ドロップ操作（v5.2の目玉機能）

#### 🖱️ 画像のドラッグ操作
1. **ドラッグ開始**: 画像サムネイルをクリックして少し動かす
2. **ゴーストウィンドウ表示**: 👻 ドラッグ中の画像プレビューが表示
3. **複数ファイル対応**: 複数選択してからドラッグすると「📁 N個のファイル」表示
4. **ドロップ先指定**: 目的のフォルダにドロップ

#### 📁 保存先フォルダ設定の簡単化
- **従来の方法**: 「保存先選択」ボタンでフォルダを指定
- **🆕 新方法**: フォルダを**保存先入力欄に直接ドラッグ&ドロップ**

#### 🎬 ドラッグ&ドロップの詳細な流れ
1. **画像を選択**: チェックボックスで1つまたは複数を選択
2. **ドラッグ開始**: 選択した画像のいずれかをドラッグ
3. **ゴースト表示**: 透明度0.8でサムネイルまたはファイル数が表示
4. **フォルダにドロップ**: エクスプローラー or 保存先入力欄にドロップ
5. **アクション選択**: 「コピー」「移動」「キャンセル」のダイアログ表示
6. **進捗表示**: 「N / M」形式の進捗ダイアログでリアルタイム表示
7. **完了通知**: 「N個のファイルをコピー/移動しました」

#### ⚙️ ドラッグ&ドロップの詳細設定
設定ファイル（app_config.json）で調整可能：
```json
{
  "enable_drag_ghost": true,          // ゴーストウィンドウの有効/無効
  "drag_ghost_opacity": 0.8,          // ゴーストの透明度（0.0-1.0）
  "drag_threshold_pixels": 5          // ドラッグ開始距離（ピクセル）
}
```

## 🛠️ 8. トラブルシューティング（v5.2対応版）

### ❌ 8.1 ドラッグ&ドロップ関連のトラブル

#### tkinterdnd2のインストールエラー
**症状**: `pip install tkinterdnd2` でエラーが発生

**よくあるエラーメッセージ**:
- `Could not find a version that satisfies the requirement tkinterdnd2`
- `No matching distribution found for tkinterdnd2`

**対処法**:
1. **pip のアップデート**:
   ```cmd
   pip install --upgrade pip
   ```

2. **バージョン指定でのインストール**:
   ```cmd
   pip install tkinterdnd2>=0.3.0
   ```

3. **代替パッケージの使用**:
   ```cmd
   pip install tkinterdnd2==0.3.0
   ```

#### ドラッグ&ドロップが機能しない
**症状**: 画像をドラッグできない、またはドロップが認識されない

**原因と対処法**:
1. **tkinterdnd2の未インストール**:
   ```cmd
   pip list | findstr tkinterdnd2
   pip install tkinterdnd2
   ```

2. **権限不足**:
   - 管理者権限でアプリケーションを実行
   - ドロップ先フォルダの書き込み権限を確認

3. **ウイルス対策ソフトの干渉**:
   - アプリケーションを例外設定に追加
   - リアルタイムスキャンを一時的に無効化

#### ゴーストウィンドウが表示されない
**症状**: ドラッグ中の視覚的フィードバックが表示されない

**対処法**:
設定ファイル（app_config.json）を確認・修正：
```json
{
  "enable_drag_ghost": true,
  "drag_ghost_opacity": 0.8,
  "drag_threshold_pixels": 5
}
```

### ❌ 8.2 インストール関連のトラブル

#### requirements.txtでのインストールエラー（v5.2版）
**症状**: `pip install -r requirements.txt` でエラーが発生

**対処法**:

## ⚙️ 9. 設定リファレンス

### 📄 9.1 app_config.json 詳細設定（v5.2版）

設定ファイルの各パラメータの詳細説明：

```json
{
  "thumbnail_size": [200, 200],
  "max_display_items": 50,
  "window_geometry": "1600x900",
  "viewer_geometry": "1200x800",
  "max_thumbnails_memory": 200,
  "thread_pool_size": 4,
  "display_batch_size": 20,
  "large_search_warning_threshold": 20000,
  "enable_predictive_caching": true,
  "predictive_pages": 1,
  "memory_cache_size": 2000,
  "enable_thumbnail_caching": true,
  "supported_formats": [".jpg", ".jpeg", ".png", ".tiff", ".webp"],
  "config_file": "app_config.json",
  
  // v5.2で新規追加：ドラッグ&ドロップ設定
  "enable_drag_ghost": true,
  "drag_ghost_opacity": 0.8,
  "drag_threshold_pixels": 5
}
```

#### 🆕 ドラッグ&ドロップ設定パラメータ

**enable_drag_ghost**: ゴーストウィンドウ表示の有効/無効
- `true`: ドラッグ中にプレビュー表示（推奨）
- `false`: ゴーストウィンドウ無効（軽量化）

**drag_ghost_opacity**: ゴーストウィンドウの透明度
- 範囲: 0.0（完全透明）〜 1.0（完全不透明）
- 推奨値: 0.8（見やすさと邪魔にならないバランス）

**drag_threshold_pixels**: ドラッグ開始の閾値（ピクセル）
- デフォルト: 5ピクセル
- 小さくすると：誤操作しやすい、敏感
- 大きくすると：確実なドラッグが必要

## ❓ 10. FAQ（よくある質問）

### Q1: v5.1からv5.2への移行時間はどのくらいかかりますか？
**A**: データベースは互換性があるため、**即座に利用可能**です：
- 設定ファイルの移行：1分未満
- 新しい依存関係のインストール：2-5分
- 初回起動：従来通り

### Q2: tkinterdnd2とは何ですか？なぜ必要ですか？
**A**: ドラッグ&ドロップ機能を実現するPythonライブラリです。

**機能**:
- 画像のドラッグ操作
- フォルダのドロップ受け入れ
- 複数ファイルの一括操作

**インストール方法**:
```cmd
pip install tkinterdnd2>=0.3.0
```

### Q3: ドラッグ&ドロップが重いです。軽量化できますか？
**A**: 以下の設定で軽量化できます：
```json
{
  "enable_drag_ghost": false,      // ゴーストウィンドウ無効
  "drag_threshold_pixels": 10      // 閾値を大きく
}
```

### Q4: ドラッグ中の画像プレビューが邪魔です
**A**: ゴーストウィンドウを無効にできます：
```json
{
  "enable_drag_ghost": false
}
```

### Q5: 複数ファイルをドラッグする時の表示を変更できますか？
**A**: 現在は固定ですが、透明度は調整可能です：
```json
{
  "drag_ghost_opacity": 0.5    // より透明に
}
```

### Q6: エクスプローラー以外にもドロップできますか？
**A**: 以下にドロップ可能です：
- Windows エクスプローラーのフォルダ
- デスクトップ
- アプリ内の保存先入力欄
- 他のファイル管理ソフト（一部）

### Q7: ドラッグ&ドロップでエラーが発生します
**A**: 以下を確認してください：
1. tkinterdnd2がインストールされているか
2. ドロップ先フォルダの書き込み権限
3. ウイルス対策ソフトの設定
4. 管理者権限での実行
