画像メタ情報検索くん v2.0 取扱説明書
1. はじめに
この度は、「画像メタ情報検索くん」をご利用いただき、誠にありがとうございます。

本アプリケーションは、お使いのコンピュータに保存されているたくさんの画像の中から、画像に含まれている「説明」や「作ったときのメモ」を頼りに、目的の画像を素早く探し出すためのソフトウェアです。

特に、AIで画像を作成した際に記録される**プロンプト（画像を作るための『呪文』や『命令文』）**をもとに検索することを得意としています。この説明書を参考に、快適な画像管理をご体験ください。

2. 導入と準備（初回のみ）
アプリケーションを初めてお使いになる際は、3つのステップで簡単な準備を行います。この作業は、一番最初の1回だけで結構です。

ステップ1：Pythonのインストール
本アプリケーションは、Python（パイソン）というプログラミング言語で動作します。

重要: もし、お使いのコンピュータにPythonがインストールされていない場合は、公式サイトからダウンロードしてインストールしてください。その際、インストーラーの最初の画面で必ず「Add Python to PATH」または「Pythonを環境変数PATHに追加する」というチェックボックスにチェックを入れてください。 これにチェックを入れることで、かんたん起動がうまく動作します。
ステップ2：ファイルの準備
デスクトップなど、分かりやすい場所に新しいフォルダを一つ作成し（例：「画像検索アプリ」）、その中に以下の3つのファイルをまとめて入れてください。

search_app.py （アプリケーション本体）
run_search_app.bat （かんたん起動ファイル）
install.bat （初回セットアップ用ファイル）
install.bat の中身は、以下の2行となります。（ご自身で作成される場合は、テキストファイルに以下を記述し、ファイル名をinstall.batとして保存してください）


pip install Pillow exifread watchdog
pause


ステップ3：必要な「部品」のインストール
アプリケーションが動くために必要な「部品」（モジュール）をインストールします。

ステップ2で準備したフォルダの中にある install.bat ファイルをダブルクリックして実行してください。
黒い画面が表示され、部品のインストールが自動的に始まります。
完了したら、「続行するには何かキーを押してください...」と表示されるので、何かキーを押して画面を閉じてください。
これで、すべての準備が完了しました。

3. アプリケーションの起動
準備がすべて完了したら、今後は run_search_app.bat ファイルをダブルクリックするだけでアプリケーションが起動します。

ターミナルを開いたり、難しい命令文を入力したりする必要はもうありません。

4. 基本的な使い方
4.1. 画面の構成
アプリケーションは、大きく4つのエリアに分かれています。

① 検索設定エリア（左上）: どこを、どんな言葉で探すかを指定します。
② 機能エリア（中上）: 履歴の呼び出しや、ファイルのコピー・移動などを行います。
③ 表示・ソート設定エリア（右上）: 見つかった画像の並べ方や表示数を決めます。
④ 検索結果エリア（下半分）: 見つかった画像がここに一覧で表示されます。
4.2. 検索の基本手順
「検索対象フォルダ」を指定します。
エリア①にある「フォルダ選択」ボタンを押し、画像がたくさん入っているフォルダを選びます。

「検索キーワード」を入力します。
探したい画像の説明やプロンプトに含まれる言葉（例: 1girl, blue_hairなど）をエリア①の入力欄に入力します。

青色の「検索スタート」ボタンを押します。
指定したフォルダ内から、キーワードに一致する画像を探し始めます。探し終わると、エリア④に画像が表示されます。

4.3. 検索結果の操作
画像の選択:
サムネイル画像（一覧の小さい画像）を1回クリックすると、その画像を選択できます（左上のチェックボックスがオンになります）。

画像の拡大表示:
サムネイル画像をダブルクリックすると、大きな画像ビューアが開き、じっくり確認できます。ビューアは矢印キーで前後の画像に移動できます。

ページの移動:
画像がたくさん見つかった場合は、自動でページが分かれます。エリア③にある「前」「次」ボタンでページを移動できます。また、数値を入力して「ページ移動」ボタンを押せば、直接そのページへジャンプできます。

4.4. ファイルのコピーと移動
コピーまたは移動したい画像をクリックして選択します（複数選択も可能）。
エリア②の「選択したファイルを...」の欄にある「保存先選択」ボタンを押し、ファイルを移したい先のフォルダを指定します。
「コピー」または「移動」ボタンを押すと、選択した画像が指定したフォルダへ複製・移動されます。
5. 便利な機能（応用編）
右クリックメニュー:
画像のサムネイルを右クリックすると、便利なメニューが表示されます。特に**「プロンプト関連」**からは、AI画像の「呪文」であるプロンプトやネガティブプロンプトを簡単にコピーでき、大変便利です。

検索オプション:
エリア①にあるチェックボックスで、より細かい検索が可能です。

AND検索: チェックを入れると、入力したキーワードがすべて含まれる画像だけを探します。
ネガティブ含む: AI画像のネガティブプロンプト（描いてほしくないものの指示）も検索対象に含めるかどうかを切り替えます。
サブフォルダも検索: チェックを外すと、指定したフォルダの直下だけを検索し、中のフォルダは無視します。
履歴とお気に入り:

一度検索した条件は「検索履歴」プルダウンに自動で保存され、後からもう一度検索できます。
よく使う検索条件は「お気に入り設定保存」ボタンで保存でき、次回アプリ起動時に自動で読み込まれます。
表示のカスタマイズ:
エリア③で、見つかった画像の並び順（ファイル名順、更新日時順、解像度順など）や、1ページに表示する画像の数を自由に変更できます。

6. うまく起動しない場合
もしrun_search_app.batをダブルクリックしても起動しない場合は、以下の点をご確認ください。

「'python' は...認識されていません。」と表示される場合

原因: Pythonのインストール時に、「Add Python to PATH」にチェックを入れ忘れた可能性が高いです。
対策: お手数ですが、一度Pythonをアンインストールし、再度インストールする際に必ずチェックを入れるようにしてください。
「ModuleNotFoundError」というエラーが表示される場合

原因: 必要な部品のインストールが正常に完了していません。
対策: もう一度 install.bat ファイルをダブルクリックして、部品のインストールを試みてください。
以上で、取扱説明書は終了です。このアプリケーションが、あなたの創作活動や画像管理の一助となれば幸いです。
